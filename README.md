# 🛒 E-Commerce Sales Data Analysis

## 📌 Project Overview
This project analyzes e-commerce sales data to identify revenue trends, top-performing products, and customer purchasing behavior.

## 🛠 Tools Used
- SQL
- MS Excel
- Power BI

## 📊 Dataset Description
The dataset includes:
- Order details
- Product categories
- Sales amount
- Customer information
- Order dates

## 🔎 Analysis Performed
- Revenue trend analysis by month
- Top product category identification
- Regional sales distribution
- KPI tracking using Excel Pivot Tables
- Interactive dashboard creation

## 📈 Key Insights
- Identified seasonal sales peaks
- Top 20% products contributing major revenue
- Regional revenue concentration patterns
- Customer purchase frequency trends
